# radar2

MATLAB simulation project for radar scenario modeling (birds, quadcopters, fixed-wing UAVs, ground vehicles).

Main application: [`BirdScenarioMVP`](BirdScenarioMVP/README.md)

## Quick start

```matlab
cd('BirdScenarioMVP')
main
```

## Tests

```matlab
cd('BirdScenarioMVP')
runAllTests
```
