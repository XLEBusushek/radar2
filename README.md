# radar2

MATLAB-проект моделирования радиолокационных сценариев (птицы, квадрокоптеры, БПЛА с неподвижным крылом, наземные транспортные средства).

Основное приложение: [`BirdScenarioMVP`](BirdScenarioMVP/README.md)

## Быстрый старт

```matlab
cd('BirdScenarioMVP')
main
```

## Тесты

```matlab
cd('BirdScenarioMVP')
runAllTests
```
